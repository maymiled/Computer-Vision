# Computer Vision – Lab Submissions

## 📂 GitHub Repository (Mandatory)

Each student must create **one GitHub repository** that will be used for **all Labs during the semester**.

Inside the repository, create **one folder per LAB**.

### Required structure:
```text
your-repo/
├── Lab1/
├── Lab2/
├── Lab3/
├── Lab4/
├── Lab5/
├── Lab6/
└── README.md
```

⚠️ Do **not** create a new repository for each LAB.

---

## 📤 Submission Rules

- Each lab must be placed in its corresponding folder (`Lab1`, `Lab2`, etc.)
- Push your work **regularly** to GitHub
- The **commit history** will be used to track progress
